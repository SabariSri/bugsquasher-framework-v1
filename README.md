# sample-mvn-framework-v1

Pre-Requisite
1. Install Java
2. Install Node.js
3. Install Appium through Node.js
4. Install Android SDK
5. Set the environment variables with ANDROID_HOME 

Integrated the following,
1. Selenium
2. TestNG
3. Maven
4. Extent Reports
5. Apache POI
6. RestAssured
7. mWeb config and functions
